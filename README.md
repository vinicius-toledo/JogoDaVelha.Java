# 🕹️ Jogo da Velha com IA Defensiva (Java)

Este é um projeto de console desenvolvido em Java que implementa o clássico Jogo da Velha. 
O diferencial deste projeto é a implementação de uma **IA Defensiva** básica.

## 🚀 Funcionalidades
- **Modo Humano vs IA**: Jogue contra o computador.
- **IA Inteligente**: O computador analisa o tabuleiro e bloqueia as suas jogadas se você estiver prestes a vencer.
- **Interface via Terminal**: Visualização simples e limpa diretamente no console.

## 🛠️ Tecnologias Utilizadas
- **Java JDK 17+**
- **Matrizes (Arrays bidimensionais)**: Para a lógica do tabuleiro.
- **Algoritmo de Simulação**: A IA testa as jogadas do usuário antes de decidir a sua própria.

## 📖 Como Rodar o Projeto
1. Clone o repositório.
2. Abra o ficheiro `JogoDaVelha.java` no seu terminal ou IDE (IntelliJ/VS Code).
3. Compile o código: `javac JogoDaVelha.java`
4. Execute: `java JogoDaVelha`
